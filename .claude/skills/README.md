# Skills для Binance Trading System

Эта директория содержит custom skills для автоматизации разработки.

## Структура

```
.claude/skills/
├── README.md              # Этот файл
├── elixir/                # Elixir-специфичные skills
├── phoenix/               # Phoenix framework skills
├── binance/               # Binance API skills
└── database/              # Database skills
```

## Использование Skills

### В Claude Code:

```bash
# Список доступных skills
/skills list

# Использовать skill
/skill skill-name
```

## Доступные Skills

После распаковки архива, здесь появится список всех доступных skills.
